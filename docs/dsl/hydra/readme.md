# Hydra grammar

The Hydra subgrammar extends the [core grammar](../core) by adding terms specific
to the [Hydra specification](http://www.hydra-cg.com/spec/latest/core/). 

Hydra scenarios are text files with `.hydra` extension.

## Supported steps

* [Operation](./operation)
* [Type](./type)

## Helper statements

* [Prefix](./prefix)
* [Invoke](./invoke)
